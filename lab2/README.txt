В папке "src" находятся все файлы с исходным кодом 
Всего в папке "src" 6 файлов:
1) 2 для первого степа - step1_source.cpp, step1_test.cpp, functions_step1.h, программа, тест и реализация алгоритма в заголовочном файле соответственно;
2) 2 для второго степа - step2_source.cpp, step2_test.cpp, functions_step2.h, программа, тест и реализация алгоритма в заголовочном файле  соответственно; 

При сборке программы сначала запускается cmake ("cmake CMakeLists.txt"), затем просто make (уже создастся Makefile).

После команды make появятся 4 исполняемых файла:
1) два из них - файлы, запускающие программу, называются runStep1 и runStep2;
2) остальные два - файлы, запускающие тесты к программам каждого степа,
   runStep1Tests и runStep2Tests соответственно.

Программа первого степа - поиск указанного количества шаблонов в заданной строке
Программа второго степа - поиск подстроки с джокером в заданной строке
